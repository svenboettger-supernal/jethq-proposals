Workspace · 09

Sales Ops

The existing Yearly Results section, made real-time and connected to Salesforce. Inbound leads, pipeline, forecast, comp, leaderboards, and deal-health flags in one place, with listings sourced only from the Hub.

01 / Overview

The brief

Salesforce is used the traditional way, deals, contacts, and companies. The CRM manager can grant API access, so Flight Deck pulls the data in and visualizes it, cleaner and more relevant than the raw Salesforce views.

One concern to resolve. A separate outsourced integration is being built between JetNet and Salesforce that duplicates the Hub's existing integration and creates two places where available listings appear. Flight Deck keeps listings in the Hub only, so there is a single source. Sales Ops visualizes the sales picture, it does not become a second listings database. Whether Salesforce is retired later is a separate, future decision.

02 / Objectives

What success looks like

  • Real-time results replace the static Yearly Results, so the sales picture is current the moment it is opened.
  • Inbound leads are tracked with source attribution and a running log of every touch.
  • Pipeline is pulled from Salesforce by API and visualized cleaner than the raw CRM views.
  • Deal-health flags surface deals that are stalling or at risk before they slip.
  • Comp and leaderboards keep performance visible against goal.
  • Forecast rolls up from the live pipeline instead of a manual spreadsheet.
  • Listings are referenced from the Hub only and never duplicated inside Sales Ops.
