Assistant

AI Helper

An in-platform assistant for the everyday language work the team already does. Translation, grammar, and drafting inside Flight Deck, so the team stops leaving the platform. A private-model path keeps sensitive legal work local.

01 / Overview

The brief

The team already pays for ChatGPT and uses it for translation, grammar, and cleanup. A document gets translated into Brazilian Portuguese, then a native speaker refines it. That work happens in a separate tab today. An in-platform helper keeps it inside Flight Deck, so the team stops toggling out to get through the same language tasks it does every day.

We will be honest about the value. A JetHQ-owned assistant will not beat the public models at open-ended tasks. Its value is integration and data isolation. The private-model path runs locally under a stronger data protection agreement, so sensitive data never leaves Supernal servers, and it can replace a per-matter legal tool such as GCAI that charges by matter. This module is kept separate from the core build so it never blocks it. Timing is to be confirmed.

02 / Objectives

What success looks like

  • An in-platform assistant for translation, grammar, cleanup, and drafting.
  • A translation flow the team can trust for a first pass before a human refines it.
  • The assistant available inside Flight Deck, not in a separate tab.
  • A private-model path that keeps sensitive data local under a stronger data protection agreement.
  • The option to retire a per-matter legal tool such as GCAI that charges by matter.
  • A clear boundary so this never blocks the core Flight Deck build.
